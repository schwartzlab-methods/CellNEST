## Relay network confidence scoring

### Requirements

This step requires the `*_relay_pattern_count.csv` file generated by `relay_extract`.

### Description

For each relay network (`ligand_1–receptor_1 to ligand_2–receptor_2`), CellNEST queries the curated protein-protein interaction and transcription factor-target gene databases to determine whether a signalling path exists from `receptor_1` to a transcription factor that activates `ligand_2`. 

The combined confidence score is the product of the cumulative protein-protein interaction edge weights along this path and the confidence score of the terminal transcription factor.

### Example command

```bash
cellnest relay_confidence \
    --input_path='output/V1_Human_Lymph_Node_spatial/CellNEST_V1_Human_Lymph_Node_spatial_relay_pattern_count.csv' \
    --output_path='output/V1_Human_Lymph_Node_spatial/relay_confidence_score_for_top3kCCC.csv' \
    --organism='human' \
    --database_dir='database/'
```

### Arguments

| Argument | Description |
|---|---|
| `--input_path` | Path to the `*_relay_pattern_count.csv` file from `relay_extract` |
| `--output_path` | Path for the output confidence score CSV |
| `--organism` | Organism profiled in the experiment; either `human` or `mouse` |
| `--database_dir` | Directory containing the protein-protein interaction ({organism}_signaling_ppi.csv) and transcription factor–target ({organism}_tf_target.csv) database files |

### Output

The output CSV contains one row per relay network with the following columns:

| Column | Description |
|---|---|
| `relay_pattern` | Relay network |
| `receptor_1` | Source node queried in the protein-protein interaction graph |
| `ligand_2` | Target gene queried in the GRN |
| `combined_score` | Confidence score (`None` if no path found) |
| `transcription factor` | Terminal transcription factor in the best-scoring path |
| `path` | Path from `receptor_1` to the transcription factor, or a diagnostic message if no path was found |
